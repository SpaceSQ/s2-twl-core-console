**【The Blueprint for Silicon-Based Society in Human Habitats】**

This engineering delivery contains the complete execution framework for the **S2-SWM Asymmetric Governance Protocol**. It resolves the industry's critical "Sovereignty Conflict" by subordinating Guest Embodied Robots to the Indoor Lord Agents (BMS/Digital Humans).

## 🚀 Architecture Highlights
1. **Spatio-Temporal Visas (`visa_manager.py`)**: Replaces permanent network access with dynamic, grid-restricted, time-bound spatial visas.
2. **Negotiation API (`lord_brain.py`)**: Strips robots of hardware control. Environmental changes (14D tensors) must be negotiated and approved by the Lord Agent based on human context.
3. **Immutable Spatial Ledger (`spatial_ledger.py`)**: Provides cryptographic liability tracing for all physical interactions generated by dynamic objects (TDOG).
4. **L0 Emergency Override (`lord_brain.py`)**: Bypasses software layers to enact immediate physical sanctions upon restricted zone breaches.
